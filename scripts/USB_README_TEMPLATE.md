# Babi E-book Portal (USB Edition)

ยินดีต้อนรับสู่ Babi E-book Portal แบบพกพา

## การเตรียม USB (สำคัญ)
เพื่อให้สามารถใช้งานได้ทั้งบน Windows และ macOS และรองรับไฟล์ขนาดใหญ่ **แนะนำให้ Format USB เป็นระบบ "ExFAT"**

*   **ExFAT**: (แนะนำ) อ่าน/เขียน ได้ทั้ง Windows และ Mac, รองรับไฟล์ใหญ่กว่า 4GB
*   **FAT32**: อ่าน/เขียน ได้ทั้งคู่ แต่รับไฟล์ใหญ่สุดได้แค่ 4GB (ไม่แนะนำถ้ามีหนังสือเล่มใหญ่)
*   **NTFS**: Mac อ่านได้แต่เขียนไม่ได้ (ใช้งานไม่ได้)
*   **APFS / Mac OS Extended**: Windows มองไม่เห็น (ใช้งานไม่ได้)

## โครงสร้างโฟลเดอร์ใน USB
เมื่อคุณติดตั้งลงใน USB แล้ว โครงสร้างของไฟล์จะเป็นดังนี้:

```
[Drive USB]:
  ├── README.md                  (ไฟล์คู่มือฉบับนี้)
  ├── data/                      (โฟลเดอร์เก็บข้อมูล - ใช้ร่วมกันทั้ง Mac และ Windows)
  │    └── babi.db               (ฐานข้อมูลหนังสือที่เตรียมไว้ให้แล้ว)
  ├── uploads/                   (โฟลเดอร์เก็บไฟล์หนังสือและรูปปก - ใช้ร่วมกัน)
  │    └── [ID หนังสือ] 
  |        (ข้อมูลหนังสือแต่ละเล่ม)
  ├── Windows/                   (สำหรับใช้งานบน Windows)
  │    └── Babi E-book Portal.exe
  └── Mac/                       (สำหรับใช้งานบน macOS)
       └── Babi E-book Portal.app
```

## วิธีการใช้งาน

### สำหรับ Windows
1. เข้าไปที่โฟลเดอร์ `Windows`
2. ดับเบิ้ลคลิกที่ `Babi E-book Portal.exe`

### สำหรับ macOS
1. เข้าไปที่โฟลเดอร์ `Mac`
2. ดับเบิ้ลคลิกที่ `Babi E-book Portal.app`

## การย้ายข้อมูล
หากต้องการย้ายข้อมูลหนังสือไปยัง USB อื่น หรือสำรองข้อมูล
ให้ทำการก๊อปปี้โฟลเดอร์ `data` และ `uploads` ไปด้วยเสมอ

---
**หมายเหตุ:** โปรแกรมจะทำงานได้ถูกต้องเมื่อโครงสร้างโฟลเดอร์เป็นไปตามที่ระบุไว้ข้างต้นเท่านั้น
ห้ามเปลี่ยนชื่อหรือย้ายตำแหน่งโฟลเดอร์ `data` และ `uploads`
