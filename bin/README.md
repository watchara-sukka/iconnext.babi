# Portable Node.js Binaries

To make this application portable, you need to download the Node.js binaries for each platform and place them here.

## Windows
1. Download `node.exe` (Windows Binary) from [nodejs.org](https://nodejs.org/en/download/prebuilt-binaries).
2. Place `node.exe` in `bin/win/`.

## macOS
1. Download the macOS binary (tar.gz), extract it.
2. Place the `node` executable in `bin/mac/`.
